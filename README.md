# MSG - Mangpage(like) static Site Generator

Project that turns mangpage like documents into static HTML sites.

![Example of turning msg.1 into HTML using this program](./example.png)

## Usage

```
$ cc -o msg msg.c
$ ./msg something.1 > something.html
```
